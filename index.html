Purpose

This repository is a Chartered CASE Workspace supporting engineering collaboration between Polymorphic and PublicLogic. It contains the authoritative governance models, lifecycle definitions, and system constraints required to implement VAULT-aligned municipal workflows safely and defensibly.

This is not a demo repository, a backlog, or an exploratory prototype. It is a governed workspace in which scope, authority, and lifecycle rules are explicit.
