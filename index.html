VAULT Frameworks (Implemented as Modules)

Each VAULT framework defines the governance, authority, lifecycle, and enforcement rules for a municipal function. These frameworks are implemented by Polymorphic as runtime modules, but their structure and constraints are defined here and do not change by technology choice.

  • PRR Framework — Public Records (Implemented as Module)
  • CLERK Framework — Licensing and Permitting (Implemented as Module)
  • FISCAL Framework — Finance and Procurement (Implemented as Module)
  • FIX Framework — Maintenance and Work Orders (Implemented as Module)
  • TIME Framework — Timesheets and Payroll (Implemented as Module)
  • ONBOARD Framework — Employee Onboarding (Implemented as Module)

Frameworks define what must be true. Modules define how it is executed.
