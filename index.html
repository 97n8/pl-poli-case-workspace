Polymorphic × PublicLogic

Engineering CASE Workspace
Status: Active · Authority Bound · Canonical

Purpose

This repository is a Chartered CASE Workspace governing the engineering collaboration between Polymorphic and PublicLogic.

It contains the authoritative governance models, lifecycle definitions, and system constraints required to implement VAULT-aligned municipal workflows safely, defensibly, and with continuity through turnover.

This is not a backlog, a demo, or an exploratory prototype.
Work that is not represented here is out of scope by definition.

How to Use This Workspace

  1. Read Read First
  2. Review CASE Workspace Definition
  3. Confirm Authority and Scope
  4. Understand constraints in What Is Non-Negotiable
  5. Then proceed to module-level materials

Canonical CASE Lifecycle

All work governed by this CASE follows the lifecycle below. Implementations may vary, but this structure must be satisfied.
